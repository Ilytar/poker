1. Данный проект - попытка сделать аналог покера из игры "Ведьмак 1" и улучшить свои навыки в React. Для разработки использовались такие библиотеки как React, React-three-fiber, Cannon.js, Swiper, Redux. Также использовался препроцессор SASS и scss-модули.
2. Все 3д-модели были созданы в Blender'е и экспортированы в формат gltf. Для моделей игральной доски и кубиков использовались оригинальные текстуры с небольшими доработками. Модели можно найти в моём публичном репозитории: https://github.com/Ilytar/models_poker , откуда они и загружаются в игру.
3. Часть изображений была получена из скриншотов оригинальной игры, а другая часть с сайта: https://witcher.fandom.com/wiki/The_Witcher_dice_poker
4. В игре в левом нижнем углу есть кнопка, которая вывоводит на экран модальное окно с описанием игры.
5. Есть репозиторий с уже готовой сборкой данной игры: https://github.com/Ilytar/poker_build
